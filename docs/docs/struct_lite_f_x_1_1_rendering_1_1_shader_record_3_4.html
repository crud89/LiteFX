◆ localData()

const void * LiteFX::Rendering::ShaderRecord<>::localData ( ) const
inlineconstexproverridevirtualnoexcept

Returns a pointer to the shader-local data of the record.Shader-local data is a piece of constant data that is available to the shader during invocation. During a ray hit/miss event, the shader record is selected based on geometry (IBottomLevelAccelerationStructure), instance (ITopLevelAccelerationStructure and an implementation-specific offset. The selected record is then used to load the shader and pass the shader local data to it.
