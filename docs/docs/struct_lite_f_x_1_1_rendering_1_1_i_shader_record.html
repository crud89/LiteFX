Detailed Description

Describes a record within a shader binding table.

A shader record contains a shader group, that is either a single shader of type ShaderStage::RayGeneration, ShaderStage::Intersection, ShaderStage::Miss or ShaderStage::Callable, or a pair of types ShaderStage::ClosestHit and ShaderStage::AnyHit, where at least one of them needs to be set.

Typically you do not want to implement this interface itself. Prefer using the ShaderRecord template to create shader records instead.

ShaderRecord, IShaderProgram::buildShaderBindingTable

virtual constexpr const void * LiteFX::Rendering::IShaderRecord::localData ( ) const
constexprpure virtualnoexcept

Returns a pointer to the shader-local data of the record.

Shader-local data is a piece of constant data that is available to the shader during invocation. During a ray hit/miss event, the shader record is selected based on geometry (IBottomLevelAccelerationStructure), instance (ITopLevelAccelerationStructure and an implementation-specific offset. The selected record is then used to load the shader and pass the shader local data to it.
