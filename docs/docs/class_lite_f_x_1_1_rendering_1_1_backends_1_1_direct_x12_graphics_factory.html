Detailed Description

A graphics factory that produces objects for a DirectX12Device.

The DX12 graphics factory is implemented using D3D12 Memory Allocator.

overridevirtual

Creates an image that is used as render target attachment.

Render target attachments are always generated in the initial state D3D12_RESOURCE_STATE_ALL_SHADER_RESOURCE and must be transitioned into D3D12_RESOURCE_STATE_RENDER_TARGET before drawing to them.

Depth/Stencil attachments are created with D3D12_RESOURCE_STATE_ALL_SHADER_RESOURCE | D3D12_RESOURCE_STATE_DEPTH_READ and must be transitioned to D3D12_RESOURCE_STATE_DEPTH_WRITE before writing to them.

The attachment type (render target or depth/stencil) is acquired from format . If the format has a depth and/or a stencil channel, it is assumed to be a depth/stencil attachment.

The initial state and source heap of a constant buffer depends on the usage parameter:

  • Staging and Dynamic: Created on the D3D12_HEAP_TYPE_UPLOAD<c> heap in the initial state D3D12_RESOURCE_STATE_GENERIC_READ. Should be transitioned to D3D12_RESOURCE_STATE_VERTEX_AND_CONSTANT_BUFFER before accessing it.
  • Resource: Created on the D3D12_HEAP_TYPE_DEFAULT<c> heap in the initial state D3D12_RESOURCE_STATE_COPY_DEST. Should be transitioned to D3D12_RESOURCE_STATE_VERTEX_AND_CONSTANT_BUFFER before accessing it.
  • Readback: Created on the D3D12_HEAP_TYPE_READBACK<c> heap in the initial state D3D12_RESOURCE_STATE_COPY_DEST. Should be transitioned to D3D12_RESOURCE_STATE_COPY_SOURCE before transferring data from it.

The initial state and source heap of an index buffer depends on the usage parameter:

  • Staging and Dynamic: Created on the D3D12_HEAP_TYPE_UPLOAD<c> heap in the initial state D3D12_RESOURCE_STATE_GENERIC_READ. Implicitly transitions to D3D12_RESOURCE_STATE_INDEX_BUFFER when first used by the GPU †.
  • Resource: Created on the D3D12_HEAP_TYPE_DEFAULT<c> heap in the initial state D3D12_RESOURCE_STATE_COPY_DEST. Implicitly transitions to D3D12_RESOURCE_STATE_INDEX_BUFFER when first used by the GPU †.
  • Readback: Created on the D3D12_HEAP_TYPE_READBACK<c> heap in the initial state D3D12_RESOURCE_STATE_COPY_DEST. Should be transitioned to D3D12_RESOURCE_STATE_COPY_SOURCE before transferring data from it.

† Note that this implicit transition does not reflect into the buffer IDirectX12Resource::state member, so be careful when transitioning the buffer into another state.
