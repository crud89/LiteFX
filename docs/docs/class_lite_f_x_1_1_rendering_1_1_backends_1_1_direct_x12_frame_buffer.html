void mapRenderTarget (const RenderTarget &renderTarget, UInt32 index) override
 Maps a render target to a frame buffer image.When calling IRenderPass::begin, passing a frame buffer, the render pass attempts to resolve all render target images. In order for this resolution to be successful, a mapping first needs to be established between the render target and the image. This method establishes this mapping.Calling this method multiple times will overwrite the mapped index.

Maps a render target to a frame buffer image.When calling IRenderPass::begin, passing a frame buffer, the render pass attempts to resolve all render target images. In order for this resolution to be successful, a mapping first needs to be established between the render target and the image. This method establishes this mapping.Calling this method multiple times will overwrite the mapped index.
