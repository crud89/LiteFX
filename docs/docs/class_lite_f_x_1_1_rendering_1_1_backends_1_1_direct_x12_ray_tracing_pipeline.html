Detailed Description

Implements a DirectX 12 RayTracingPipeline.

Note that the ray tracing pipeline does not set its handle, as it is a different base type from the one used to define ray tracing pipelines. Instead to obtain the handle, call DirectX12RayTracingPipeline::stateObject().

At some point it is expected that D3D introduces a unified pipeline architecture based on state objects. At this point, the pipeline state base object will switch to using state objects as its handle type and the stateObject method above will be deprecated.

See also
DirectX12RenderPipeline, DirectX12RayTracingPipelineBuilder
