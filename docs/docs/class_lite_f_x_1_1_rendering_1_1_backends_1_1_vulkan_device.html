void VulkanDevice::wait ( ) const
overridevirtual

Waits until all queues allocated from the device have finished the work issued prior to this point.Note that you must synchronize calls to this method, i.e., you have to ensure no other thread is submitting work on any queue while waiting. Calling this method only guarantees that all prior work is finished after returning. If any other thread submits work to any queue after calling this method, this workload is not waited on.
