Queues a present that gets executed after fence has been signaled on the default graphics queue.You can use this overload in situations where you do not have an IRenderPass or IFrameBuffer to render into before presenting. Instead, you typically copy into the swap chain back buffer images directly (image). This copy is done in a command buffer that must be submitted to the default graphics queue. The swap chain can then wait for the copy to finish before presenting it. Example scenarios where this is useful are, where you want to write to the back buffer from a compute shader, that does not have an equivalent to render passes.
