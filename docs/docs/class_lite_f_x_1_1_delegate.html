Detailed Description

template<typename TResult, typename... TArgs>
class LiteFX::Delegate< TResult, TArgs >

Represents a handler for an Event, that is assigned a unique token when created, so that it can be identified later.

Since a std::function or callable is non-comparable, it is not possible to identify and remove event handlers from an event, after registering them. To support this functionality, the event handler will be encapsulated in a delegate, which gets assigned with a token. This token is unique for the event, the delegate has been registered to.
