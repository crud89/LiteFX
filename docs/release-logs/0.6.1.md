# LiteFX 0.6.1 - Alpha 06

- Update toolchain to LLVM 21. (See [PR #197](https://github.com/crud89/LiteFX/pull/197))
- Allow to specify entry point names when compiling shaders with the shader build script and DXC. (See [PR #202](https://github.com/crud89/LiteFX/pull/202)

**🌋 Vulkan:**

- Update Vulkan SDK to 1.4. (See [PR #196](https://github.com/crud89/LiteFX/pull/196))
- Allow to supply custom instance and device extensions. (See [PR #198](https://github.com/crud89/LiteFX/pull/198))

**🐞 Bug Fixes:**

- Fixed two issues with the assets and shader build scripts: (See [PR #202](https://github.com/crud89/LiteFX/pull/202)
  - An issue where the asset build script could not copy multiple files to the same directory.
  - An issue where the shader build script did not properly pick up the `OUTPUT_NAME` target property.
